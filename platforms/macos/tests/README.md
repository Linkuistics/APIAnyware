# platforms/macos/tests/ — platform-level semantic tests

Behavioural tests of macOS API semantics (REFACTOR.md §14, §33): `api-semantics/`
(ownership, callbacks, threading, errors), `app-kinds/` (per-kind obligations),
and `fixtures/` (pasteboard, spotlight, sample documents/images). These prove the
platform spec captures real semantics, independent of any target.
